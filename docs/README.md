> **Searchable stylable select component for VueJS. This component is `renderless` so you are free to customize it however you need to!**


### Features
- Renderless
- Single and multiple mode
- Filtering
- Disabling options
- Keyboard navigation
- v-model and vuex support
- No dependencies
